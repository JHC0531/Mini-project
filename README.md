# 🌀 Verb Learning App

## 1. App Purpose

Learning sentence patterns is essential for understanding and expressing ideas in English.  
Verbs play an important role because they determine the structure of a sentence.  

This app is designed to help learners understand the appropriate use of verbs in the context of the 1st to 5th sentence patterns while also exposing them to expressions frequently used in daily life.

## 2. Target Learners

This app is intended for learners who want to study the appropriate use of verbs in different sentence patterns and improve their everyday conversation skills.

The current app is designed mainly for high school students, but it can also be used by middle school or university learners depending on the example sentences and types of verbs used.

## 3. Data or Content Used

The app uses an Excel file containing verbs categorized by sentence pattern and conversational sentences that include those verbs.

The data also includes feedback explaining which verb should be used in the blank and why it is appropriate.

## 4. What Users Can Learn or Do

Users can:

- Read the Korean translation of a sentence used in daily life
- Infer the appropriate verb that fits in the blank
- Check the correct answer
- Receive feedback explaining why the verb should be used in the blank

## 5. Classroom or Self-Study Use

This app can be used for:

- Learning verbs used in the 1st to 5th sentence patterns
- Learning the components of sentences in each sentence pattern
- Practicing the appropriate use of verbs in context
- Studying everyday conversational sentences

## 6. App Design Description

The app consists of the following features:

- Pattern-based filtering for the 1st to 5th sentence structures
- Randomized answer choices to improve learning reliability
- Immediate feedback for each answer
- A clean and distraction-free interface for focused practice

## 7. Difficulties or Limitations

Some limitations of the app include:

- The number of sentences is still limited
- More expressions actually used in daily life need to be added
- The app needs to provide questions that match each learner’s response level

## 8. Further Improvement

This app could be improved by adding:

- Sentence pronunciation audio
- More various example sentences and verbs
- Level-based learning support
