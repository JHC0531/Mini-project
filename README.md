This is a simple web-based learning app designed to help Korean middle/high school students master English verb patterns (1–5 sentence structures).

The app presents each question with a Korean meaning and an English sentence side by side, allowing learners to understand the meaning first and then focus on choosing the correct verb. Each item is carefully structured to reinforce sentence patterns through repeated exposure and practice.

# Key features include:

- Pattern-based filtering (1st–5th sentence structures)
- Randomized answer choices to improve learning reliability
- Immediate feedback for each answer
- Clean and distraction-free interface for focused practice

This project was built using Streamlit and is intended for classroom use, especially in environments where students are still developing basic grammatical awareness.
